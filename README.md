# 🎵 MP3 Audio Editor

Electronベースのシンプルで使いやすいMP3音声編集ツールです。音声ファイルの切り取り、音質変更、ボリューム調整が直感的なGUIで簡単に行えます。

![MP3 Audio Editor](https://img.shields.io/badge/Electron-v28-47848F?style=flat-square&logo=electron)
![Node.js](https://img.shields.io/badge/Node.js-Required-339933?style=flat-square&logo=node.js)
![License](https://img.shields.io/badge/License-MIT-blue?style=flat-square)

## ✨ 主な機能

- **🎯 時間指定による切り取り** - 開始時間と終了時間を秒単位で指定して音声を切り取り
- **🎚️ 音質調整** - 64kbps～320kbpsまでのビットレート選択が可能
- **🔊 ボリューム調整** - 0%～200%の範囲で音量を自由に調整
- **▶️ プレビュー再生** - 編集前に音声を確認できる内蔵プレイヤー
- **📊 進行状況表示** - 処理中の進捗をリアルタイムで表示
- **🎨 モダンなUI** - 美しいオレンジのグラデーションデザイン

## 📋 必要要件

- Node.js (v14以上推奨)
- npm または yarn

## 🚀 インストール

1. リポジトリをクローンまたはダウンロード

```bash
git clone <repository-url>
cd audio-edit-tool
```

2. 依存関係をインストール

```bash
npm install
```

依存パッケージ:
- `electron` - デスクトップアプリケーションフレームワーク
- `fluent-ffmpeg` - FFmpegのNode.jsラッパー
- `ffmpeg-static` - FFmpegバイナリ（自動で含まれます）

## 💻 使い方

### アプリケーションの起動

```bash
npm start
```

### 基本的な使い方

1. **ファイルを選択**
   - 「MP3ファイルを選択」ボタンをクリック
   - 編集したい音声ファイル（MP3, WAV, OGG, M4A）を選択

2. **切り取り範囲を設定**
   - 開始時間と終了時間を秒単位で入力
   - 切り取り時間がリアルタイムで表示されます

3. **音質を選択**
   - ドロップダウンから希望のビットレートを選択
   - 64kbps（低品質）～320kbps（最高品質）

4. **ボリュームを調整**
   - スライダーで音量を調整（0%～200%）
   - 100%が元の音量です

5. **保存**
   - 「保存」ボタンをクリック
   - 保存先を指定して処理を実行

## ⚙️ ビットレートオプション

- 64 kbps - 低品質（ファイルサイズ小）
- 96 kbps - 中低品質
- 128 kbps - 標準品質（デフォルト）
- 192 kbps - 高品質
- 256 kbps - より高品質
- 320 kbps - 最高品質（ファイルサイズ大）

## ⚠️ 注意事項

- 大きな音声ファイルの処理には時間がかかる場合があります
- ボリュームを200%に設定すると音割れする可能性があります
- 著作権で保護された音声ファイルの取り扱いには十分注意してください

## 🐛 トラブルシューティング

### アプリが起動しない
- Node.jsが正しくインストールされているか確認
- `npm install`を再実行してみてください

### 音声処理が失敗する
- 入力ファイルが破損していないか確認
- 十分なディスク容量があるか確認
- 終了時間が開始時間より後に設定されているか確認

### 音質が悪い
- より高いビットレート（192kbps以上）を選択してください
- 元の音声ファイルの品質以上には改善できません
